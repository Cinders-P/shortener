URL Shortening API

Example Output
{"original": "https://docs.mongodb.com/","short": "https://sh0.herokuapp.com/0"}

Append a valid url onto the https://sh0.herokuapp.com/new/ endpoint to return a shortened form. Trying to access the shortened URL will redirect back to the original page. Try it out by first accessing https://sh0.herokuapp.com/new/https://www.freecodecamp.com/.
